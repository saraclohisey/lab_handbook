# SOP_naming

# Aim
The aim of this SOP is to ensure consistency in naming conventions across the lab.

# Method
Naming changes depending on what is being named.


### Experiment
- All experiments should be numbered and recorded in your lab book.

#### YY_proj_01
- 25 : Year
- proj : project shorthand
- 01 : experiment_number


### Oligo
- All oligos should be recorded in an excel spreadsheet 
- Oligo spreadsheets should be collated in a communal space (Sharepoint/inventories)

#### oXX0000 eg: oSC1234
 - o : oligo
 - XX : author initials
 - Number 3-4 digits: 0000

#### Headings for oligo spreadsheet
 - Oligo Number
 - Application
 - Sequence
 - Partner
 - Tm
 - Description
 - Location
 - Notes

### Plasmid

- All plasmids should be recorded in an excel spreadsheet 
- Plasmid spreadsheets should be collated in a communal space (Sharepoint/inventories)

#### pXX0000 eg: pSC1234
- p : plasmid
- XX : author initials
- Number 3-4 digits: 0000

#### Headings for plasmid spreadsheet
 - Plasmid Number
 - Application
 - Backbone
 - Insert
 - Location
 - Description
 - Glycerol stock location
